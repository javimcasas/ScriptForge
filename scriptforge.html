Formato del archivo .cfg

          
Las primeras líneas son metadatos (empiezan con #). El resto es el script.

          
# name: Nombre visible en la card
# category: Management
# description: Descripción breve.

sysname {HOSTNAME}
!
interface {INTERFAZ}
 ip address {IP} {MASCARA}
